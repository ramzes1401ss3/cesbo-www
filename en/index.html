Quick Start

Input

  • DVB, ATSC, ASI
  • HTTP (MPEG-TS, HLS)
  • UDP/RTP
  • RTSP
  • MPEG-TS Файлы

Output

  • DVB: RESI DVB-C Modulator, HiDes DVB-T Modulator
  • HTTP (MPEG-TS, HLS)
  • UDP/RTP
  • MPEG-TS Files

Processing

  • Descrambling: DVB-CI
  • Scrambling: SimulCrypt, BISS
  • Demultiplexing - preparing channels for IP network
  • Multiplexing - preparing streams for DVB modulators
  • T2-MI Decapsulation
  • Analyzing receiving streams
  • Redundant inputs
  • Filtering streams and changing stream information

Other

  • Scripting language (Lua)
  • Web-Interface
  • Restrict access to the streaming content
  • Telemetry and monitoring
  • API for management
  • HbbTV
  • EPG Export (XMLTV)

Requirements

  • OS: based on the Linux kernel
  • CPU: x86, ARM, MIPS
